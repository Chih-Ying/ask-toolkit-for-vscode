January 2021 update

Welcome to the updated Alexa Skills Toolkit for January 2021 (version 2.3). This update introduces a number of new features and enhancements.

What's new?

Support self-hosted skills creation, downloading, and deployment

We support self-hosted skills in the Create a skill, Download and edit skill and Deploy skill features on the toolkit launch.

Create a skill

Developers can select "Provision your own", fill out the form and click the "Create" button to create a self-hosted skill.



Download and edit skill

Developers can download the skill package for a self-hosted skill. ASK Toolkit does not support Lambda code, which developers will have to provision separately.



Deploy skill

Developers can save changes and deploy a self-hosted skill using the "Deploy & build" button. A skill must be synchronized with changes made in the console in order to deploy. The `Skill deploy status overview` displays skill local changes and the skill remote sync status to guide developers how to successfully deploy. For self-hosted skills, it shows the skill package status only. For Alexa hosted skills, it shows both the skill package status and the skill code status.



SkillPackage Validation

The ASK Toolkit will retrieve vendor specific skill package schema once the extension is activated and there's only one skill in your workspace. The schema will be stored at YourSkill/.ask/schema, and the extension will use this schema to validate your SkillPackage structure. The validation will be triggered whenever you create / delete files under the skillPackage folder. It will also be triggered when you open the skill deploy page.
